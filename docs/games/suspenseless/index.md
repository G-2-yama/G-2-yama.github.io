# suspenseless
このゲームは一般的な推理ゲームの物語の視点を逆にして、"犯人"である主人公が"探偵"と対決する述べるゲームです。

事件現場での行動や、事件後の振る舞い、証拠品の偽装などによってストーリーが分岐するシステムで、探偵を騙し、主人公にとっての幸福なエンディングにたどり着くのがゲームの目的となります。

体験版には、物語の始まりと、主人公が事件を起こし、自分に都合のいい"真実"のため現場で行動していく部分が入っています。

お楽しみください。
